# Tiny Silent Ninja — ラフ案 3パターン

機能・UI は変更せず、キャラクターの方向性だけを比較するための案です。

## 共通仕様（全案）

| 項目 | 方針 |
|------|------|
| 頭身 | 2〜3頭身 |
| 配色 | 黒〜ダークグレー + 赤差し色 |
| 目 | 小さく光る程度（スリット or 点） |
| 刀 | 小さく、主張弱め |
| 雰囲気 | かわいい 60% / クール 40% |
| 最重要 | **idle で止まっている時に可愛い** |

---

## 案 A — 「まるっこシルエット」 *(Round Mochi)*

**キーワード:** 柔らかい・住み着き・呼吸 idle 向き

- 頭がやや大きめの**丸いフード**（顔はほぼ見えない）
- 体は**短い台形**、足はスタブ（2〜3頭身の定番）
- 赤は**額のハチマキ**のみ（情報量最小）
- 目はフードの影の中に**2つの小さな光点**
- 刀は腰に**細い1本線**程度

**強み:** 小さくても「キャラ」と分かりやすい。idle の上下呼吸が映える。  
**弱み:** ややポップ寄り（子供っぽくならないよう目を小さく保つ必要あり）

**おすすめ:** 「デスクトップに住んでいてほしい」感を最優先する場合

---

## 案 B — 「影の楔」*(Shadow Wedge)*

**キーワード:** クール寄り・忍者らしい・軽い walk 向き

- **尖ったフード**と細身のシルエット（楔形）
- 赤は**スカーフの端**だけ（流れは walk / smoke 時に活かす）
- 目は**横一文字のスリット**＋微光
- 刀は細長く、**斜め下**に控えめ

**強み:** 忍者・忍者の軽さが一目で伝わる。海外向けのクールさ。  
**弱み:** 小さすぎると尖りが潰れやすい → 48px 以上で見る前提

**おすすめ:** 「静かに見守る」クール寄り 40% を厚くしたい場合

---

## 案 C — 「座りベース」*(Sit-First)*

**キーワード:** 落ち着き・休憩感・将来の sit / smoke 向き

- **やや低い重心**（頭 : 胴 : 脚 ≒ 2 : 2 : 1.5）
- マスクは**丸みのある下半分カバー**、上に小さな目
- 赤は**帯（おび）の結び目**一点
- 刀は**短刀**サイズで体の横に収まる

**強み:** sit idle が自然に見える。派手さゼロで「見守り」タイプ。  
**弱み:** 立ち walk 時はやや地味（動きで補う設計）

**おすすめ:** Phase 2 の sit / smoke を見据え、**止まっている時**を最重視する場合
