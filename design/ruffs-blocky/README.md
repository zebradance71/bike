# Blocky Ninja — idle ラフ 3 案

**最優先:** 24px で「忍者」と一目で分かる（馴染むことより識別）  
**方向:** Minecraft / tiny indie / blocky companion  
**NOT:** 敵キャラ・ボス感・情報過多・R1 のまま再掲

walk / sit / 実装は **採用後**。

---

## 案 B1 — Headband Hero（帯が主役）

**記号の置き方:** 赤ハチマキが **いちばん目に入る**

- フード上 1/3 を **太めの赤帯**（3px 高）で横断
- その下に **黒フード塊**＋ **2点の目**（離し気味）
- 体は **6×5 の1矩形**、足は **2本の短い柱**

| 強み | 24px で「忍者」と最短で結びつく |
| 弱み | 帯が強い → キャラより「アイコン」寄り（意図的） |

**向き:** 「何のキャラか」最優先。companion は帯＋小ささで柔らげる。

---

## 案 B2 — Block Hood（インディー定番塊）

**記号の置き方:** **四角い頭＋黒フード＋帯** の三段

- 頭 = **8×8 近い正方形**（インディー/Minecraft 感）
- 額に **赤帯 2px**、顔中央に **横スリット 3px**（目の代わり）
- 胴・足は **最小矩形** のみ（合計マス数少）

| 強み | blocky がはっきり。indie companion 感 |
| 弱み | スリットが潰れると「黒四角」→ 48px 併用推奨 |

**向き:** 「端に住んでる小さなブロック忍者」

---

## 案 B3 — Slit Silhouette（スリット忍者）

**記号の置き方:** **フード＋目のスリット** が忍者の決め手

- 頭部は **上広下狭の黒塊**（くさび）
- 赤帯は **細めだがコントラスト強**（#c85848）
- 目は **横一線スリット**（2点ではなく1本線）— 忍者映画の記号
- 肩を **1px 出して** シルエットを人型に

| 強み | 「忍者」記号が文化的に強い。敵感を帯で抑えられる |
| 弱み | スリットのみだと感情は弱い → companion は色温度で調整 |

**向き:** クール寄りだが **認識速度** は B1 に次ぐ
