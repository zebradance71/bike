# Retro Desktop Companion — ラフ 3 案

**方向:** SNES / GBA 系 pixel ninja × blocky × 24〜48px で読める  
**NOT:** LEGO 風・3D・高ディテール・AIマスコット・モバイルUIキャラ

現行（丸シルエット SVG）の**完全否定ではなく**、retro / blocky への寄せ案です。  
**実装はまだ行いません。** 採用案が決まってからスプライト差し替えします。

---

## 共通ルール（全案）

| 項目 | 方針 |
|------|------|
| サイズ設計 | **24×28px** グリッド基準 → 48px は整数倍拡大 |
| 色数 | 4〜5色（黒・ダークグレー・赤・目・任意で影1） |
| ハチマキ | 赤（残す） |
| 目 | 2点のみ、発光・グラデ・ぼかしなし |
| 3D | なし（フラット、上面光なし） |
| 刀 | 1〜2px 線 or 省略可 |

---

## 案 R1 — 「GBA Dot」（まるブロック）

**キーワード:** cozy / 丸みは**ピクセル単位**だけ / いちばん「住み着き」

- 頭は **角丸ではなく階段状** の 3〜4 段で丸く見せる
- 体は **台形ブロック**（上広下狭）
- 足は **2px 幅の短い柱**（walk 向き：左右交互が映える）
- sit：足を横棒1本＋胴体沈下で「しゃがみ」

**強み:** 24px でも顔のブロックがはっきり。かわいさ寄り（60%）を維持しやすい。  
**弱み:** やや「もち」に見える可能性 → 足を R1 だけ 1px 長く調整可。

**おすすめ:** 現行の「かわいい常駐」を残しつつ retro に寄せたい場合。

---

## 案 R2 — 「SNES Chunk」（角ばりシルエット）

**キーワード:** クール寄り強め / 忍者の**くさび影**

- フード先端に **1px の尖り**（三角形ブロック）
- 胴体は **矩形＋斜めカット1マス** で楔形
- スカーフは **赤 2〜3 マス** のみ（横ににょき）
- 目は **離し気味の 1px ドット**（silent 感）

**強み:** 「レトロゲーム忍者」が一番伝わる。walk のストライドが映える。  
**弱み:** 24px で尖りが潰れると硬い → 48px 常駐前提で最適。

**おすすめ:** retro desktop companion の**主軸**にしたい場合。

---

## 案 R3 — 「Desktop Icon」（極小アイコン）

**キーワード:** 情報量最小 / **24px 最優先**

- 頭＝**8×7 マスの単一シルエット**（目と帯だけ別色）
- 体＝**6×5**、足＝**合計 4 マス**
- 刀は **表示しない** or 1 マス点
- sit：頭を 1 マス下げ、足を消して「座りブロック」

**強み:** タスクバー横・極小ウィンドウでも忍者と分かる。邪魔にならない。  
**弱み:** ディテール少なめ。感情は「静か・無表情」寄り。

**おすすめ:** 存在感を**最薄**にし、レトロ OS マスコット感を最優先する場合。
