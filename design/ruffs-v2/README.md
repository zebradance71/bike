# キャラ設計リセット — idle 超ラフ 5 案

**前提（前回の学び）**
- 24px で「忍者」と一瞬で分かる
- 赤ハチマキ・刀なし
- 敵ではなく端に住む相棒
- 動いて完成（今回は **idle のみ** ラフ）

**今回の軸**
- 欲しくなる / 消したくない
- FEZ・Crossy Road・cozy indie NPC 系（retro すぎない）
- blocky OK、ただし **別キャラ感を避ける**

完成度不要。採用後に idle 実装 → walk / sit で「動いて完成」。

---

## C1 — FEZ Buddy（ふわっと菱形頭）

- 頭が **やや菱形・ころん**（FEZ 的な「かわいい幾何」）
- **太い赤帯**が頭の「冠」ように見える
- 体は **短い台形**、足は **短いスタブ**
- 相棒感: ペットっぽい、覗いてる感

**一瞬の読み:** 帯 → 黒い丸い頭 → 目2つ → 忍者

---

## C2 — Crossy Squat（がっつり2頭身）

- **頭が体の半分**（Crossy 的なデフォルメ）
- 帯は頭の **上1/4を横切る**
- 目は **大きめ2点**（感情が出る）
- 足は **ほぼ見えない足元ブロック**

**一瞬の読み:** ちび忍者のシルエットが強い

---

## C3 — Cozy Pear（なんじゃろう感を減らす梨形）

- **上が丸い梨形フード**（敵の楔形ではない）
- 帯の下に **白っぽい目エリア**のコントラスト
- 体は **小さめ楕円塊**（indie NPC）
- 全体 **暖色寄りの赤**（相棒）

**一瞬の読み:** かわいい + 帯で忍者確定

---

## C4 — Edge Peek（端にいる感）

- **少し前のめり**（taskbar から覗く）
- 帯・目が **手前に来る** 構図
- 下は **太い安定足場**（住んでる感）
- 背中側は情報なし（シンプル）

**一瞬の読み:** 「そこにいる」+ 忍者記号

---

## C5 — Symbol Mark（記号最短）

- 情報 **最小マス数**
- **帯 + 黒三角フード + 2点** だけで忍者
- 体は **4マス程度**
- アイコンに近いが **目を入れて愛着**

**一瞬の読み:** 最速で忍者（味は薄め注意）
