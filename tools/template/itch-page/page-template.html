デスクトップに住む、小さな忍者 — {{AUTHOR}}

必要環境

  • Windows 10 / 11（64-bit）
  • 常時インターネット不要（外部通信なし）

ダウンロード

  • setup (.exe) — おすすめ（インストールするだけ）
  • portable (.exe) — 1ファイルで起動（解凍不要）
  • win64 (.zip) — Ninja2 フォルダ + README 付き
Windows SmartScreen で「不明な発行元」と出たら
「詳細情報」→「実行」 を選んでください（未署名ビルドのため表示されます）。

ZIP を使う場合

  1. ZIP を右クリック → すべて展開
  2. Ninja2 フォルダを開き README.txt を読む
  3. Ninja2.exe を実行（.dll 等は消さない）

起動確認

  • タスクバー右下（^）のトレイに忍者アイコン
  • 左クリックで表示 / 非表示、右クリックでメニュー

困ったとき

  • もう一度 exe を実行（既に起動中ならトレイを確認）
  • 再ダウンロード / 再インストール
  • 詳細: リポジトリ内 docs/user/download-and-install-ja.md
