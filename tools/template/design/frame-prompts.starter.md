## 必須 stem（テンプレ pack の `REQUIRED_STEMS`）

| stem | 用途 |
|------|------|
| idle | 待機（左下アンカー） |
| idle-vibrate-a/b | たまの振動 |
| idle-exhaust-a/b | 排気 |
| block-idle | トレイ / ランチャー向け |
| block-run-a〜d | ブロック追従ループ |

import は `scripts/characters/<id>/` に bike スクリプトをコピーしてから実行。背景は **#FF00FF のみ**（透過 PNG 禁止）。
