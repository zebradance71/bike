# Phase 2 体験チェック

機能追加なし。`npm run dev` で確認してください。

## Dev 用テスト値（本番は低頻度のまま）

| 項目 | Dev | 本番 |
|------|-----|------|
| sit 確率 | 25% | 7% |
| smoke 確率 | 18% | 4% |
| sit 時間 | 4〜8秒 | 30〜60秒 |
| 煙移動先 | 画面下・左 / 右ゾーン交互 | 画面下ランダム |

## チェックリスト

### 1. sit が可愛く見えるか
- [ ] 休んでいる感じがある（だらけすぎ・幼すぎない）
- [ ] 座り中も「忍者」として読める
- [ ] 動きが少なく、うるさくない

### 2. smoke が派手すぎないか
- [ ] 煙が小さく一瞬だけ
- [ ] チカチカ・フラッシュしない
- [ ] テレポート後に唐突すぎない

### 3. 低頻度でも「たまに動く」感（本番想定）
- [ ] ほとんど idle で、たまに歩く / 座る / 煙
- [ ] Dev で一度本番タイミングを想像できる

### 4. 常駐して邪魔に感じないか
- [ ] クリックスルーで作業の邪魔にならない
- [ ] 視界の端に留まる
- [ ] 気づかなくても問題ない

### 5. 位置が自然に見えるか
- [ ] 画面下の左右に「住んでいる」感
- [ ] 毎回同じ場所に戻りすぎない（煙後）
- [ ] タスクバー付近で不自然に重ならない
