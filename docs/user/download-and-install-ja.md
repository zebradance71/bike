# Ninja2 — ダウンロード・インストール（ユーザー向け）

## 必要環境

- **Windows 10 / 11（64-bit）**
- ARM PC（Surface Pro X 等）では **未対応**（起動時に案内が出ます）
- インターネット常時接続は **不要**（localhost のみ使用）

---

## どれをダウンロードする？

| ファイル | 向いている人 |
|----------|--------------|
| **Ninja2-x.y.z-setup-x64.exe** | **おすすめ** — インストーラ（フォルダを見ない） |
| **Ninja2-x.y.z-portable-x64.exe** | 単体 exe（解凍不要・1ファイル） |
| **Ninja2-x.y.z-win64.zip** | フォルダ版（`Ninja2/` + README 付き） |

**ZIP を展開すると `Ninja2.exe` の横に `.dll` / `.pak` などが並びます。**  
これは Discord や VS Code と同じ **Electron（Chromium）の仕様** で、exe から動かせないため **整理（サブフォルダへ移動）はできません**。散らかって見えるのは ZIP 版の限界です。普段フォルダを見たくない場合は **インストーラ** か **portable 単体 exe** を選んでください。

---

## インストーラ（setup .exe）— おすすめ

1. ダウンロードした `Ninja2-…-setup-x64.exe` を実行
2. **「Windows によって PC が保護されました」** が出たら  
   → **「詳細情報」→「実行」**（コード署名前の通常の表示です）
3. インストール先を選び **次へ → 完了**
4. 完了後 **自動起動** します（トレイに忍者アイコン）

---

## ポータブル（portable .exe）

1. `Ninja2-…-portable-x64.exe` をダブルクリック（**解凍不要**）
2. 初回は SmartScreen → **詳細情報 → 実行**

---

## フォルダ版（win64 .zip）

1. ZIP を **右クリック → すべて展開**
2. 出てきた **`Ninja2` フォルダ** を開く（中に README.txt あり）  
   ※ itch.io からダウンロードした場合も、展開後は同じく **`Ninja2/` フォルダ** になります
3. **`Ninja2.exe`** を実行  
   ※ 同フォルダの `.dll` / `.pak` は削除しないでください

---

## 起動したか確認

- タスクバー右下（^）の **トレイ** に忍者アイコン
- 左クリック → コンパニオン表示 / 非表示
- 右クリック → メニュー（Block mode、終了 など）

**何も見えないとき**

1. トレイの ^ を開く（隠れていることが多い）
2. もう一度 `Ninja2.exe` を起動（既に動いていればランチャーが前面に）
3. 再インストール / 再ダウンロード
4. ログ: トレイ → **Open user data folder** → `logs\startup.log` / `crash.log`

---

## よくあるつまずき

| 症状 | 対処 |
|------|------|
| SmartScreen で止まる | 詳細情報 → 実行 |
| zip から直接起動 | **必ず展開してから** exe を実行 |
| 2 回目クリックで反応しない | 既に起動中 → トレイを確認 |
| 忍者が動かない | トレイアイコンは出ている？ → 左クリックで表示 |
| YouTube で block しない | 拡張機能は任意。本体だけでもタイトル監視で動作 |

---

## アンインストール

**設定 → アプリ → Ninja2 → アンインストール**

設定ファイルは残る場合があります（再インストール時の利便性のため）。
